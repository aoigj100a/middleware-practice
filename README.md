# middleware-practice

時間戳記與middleware使用練習。

## Features - 產品功能
在 log 印出以下資訊 :

1. 伺服器請求的時間
2. http方法
3. 連線的url
4. 從server接收到request到server送出response所需的時間

## 環境建置與需求
1. Node.js v12.19.0
2. Express v4.17.1
3. Nodemon v2.0.6
4. luxon v1.25.0
5. response-time v2.3.2

## Installing - 專案安裝流程
1. 開啟終端機，執行以下指令 :
> git clone https://github.com/aoigj100a/middleware-practice

2. 進入專案資料夾
> cd middleware-practice

3. 安裝 npm & 透過 npm 安裝套件
> npm install

4. 使用腳本啟動專案
> npm run dev

5. 在瀏覽器進入 http://localhost:3000
